# jylee2033.github.io

Personal portfolio website for Jooyoung (Julia) Lee — Computing Science student at Simon Fraser University.

Built with [Astro](https://astro.build) and deployed to GitHub Pages.

## Tech Stack

- **Framework:** Astro 6
- **Styling:** CSS custom properties
- **Deployment:** GitHub Actions → GitHub Pages

## Local Development

```bash
npm install
npm run dev
```

## Project Structure

```
src/
├── components/   # Navbar, Footer, HeroSplit, Timeline, etc.
├── content/      # Project markdown files
├── layouts/      # BaseLayout
├── pages/        # Home, Projects, About, Contact
└── styles/       # Global CSS tokens
public/
├── images/       # Profile photo, project images
└── resume.pdf
```
